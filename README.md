Create and start a virtual enviroment \
Windows: \
`
python -m venv .venv 
` 
`
.\.venv\Scripts\activate
`
Unix: \
`
python3 -m venv .venv
` 
`
source venv/bin/activate
`

Use the following to install all required packages \
`
pip install -r requirements.txt
`
